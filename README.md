# AI Workshop

### Introduction

* This workshop focuses on **practical AI usage for developers**
* Goal: **improve productivity, reduce wasted time, and increase code quality**
* Emphasis on **real workflows**, not AI hype or demos
* AI is treated as a **developer productivity tool**, not a replacement for engineers

### Workshop Objective

* Understand where developers actually lose time
* Learn how AI tools fit into daily engineering workflows
* Use AI **deliberately**, not casually
* Improve speed **without sacrificing quality**
